# 🌌 Acerca del proyecto

**ShopCoders** es una plataforma E-Commerce moderna desarrollada especialmente para developers y entusiastas de la programación.

La tienda está enfocada en vender:

- 👕 Merchandise para programadores
- 💻 Swag tecnológico
- 🏷️ Stickers para developers
- 🎁 Accesorios geek
- 🚀 Productos inspirados en programación
- 🌐 Experiencia de compra moderna

# ⚡ Instalación

## 📋 Requisitos

- Node.js
- NPM
- MongoDB
- VS Code

---

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/IAmTamal/ShopCoders.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd ShopCoders
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Configurar variables de entorno

Crear archivo:

```bash
.env
```

Agregar:

```env
MONGODB_URI=your_database_url
NEXTAUTH_SECRET=your_secret_key
STRIPE_SECRET_KEY=your_stripe_key
```

---

## 5️⃣ Ejecutar proyecto

```bash
npm run dev
```
